207.2 W3 Create and maintain DNS zones Volver
Candidates should be able to create a zone file for a forward or reverse zone and hints for root level servers.
This objective includes setting appropriate values for records, adding hosts in zones and adding zones to the DNS.
A candidate should also be able to delegate zones to another DNS server.
