mv
El comando mv (move) permite mover, volver a nombrar un fichero o las dos cosas a la vez. 
Funciona como el comando cp.
Los parámetros -f e -i tienen el mismo efecto.
$ touch txt1 txt2 txt3
$ mv txt1 txt1.old
$ mv txt2 dir1/txt2
$ mv txt3 dir1/txt3.old
Observe la existencia del parámetro -u: si el fichero de destino es más reciente, impide que se sobreescriba.
