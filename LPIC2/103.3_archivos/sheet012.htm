rm
El comando rm (remove) suprime uno o varios ficheros y, si es preciso, una estructura completa.
rm [Opciones] fic1 [fic2...]
-i --> El comando requerirá una confirmación para cada uno de los ficheros a suprimir. Según la versión de Unix, el mensaje cambia y la respuesta también: y, Y, O, o, N, n, a veces todas.
-r --> El parámetro siguiente que se espera es un directorio. En este caso, la supresión es recursiva: se suprimen todos los niveles inferiores, tanto los directorios como los ficheros.
-f --> Fuerza la supresión.
