rmdir
El comando rmdir (remove directory) suprime uno o varios directorios. 
rmdir dir1 [dir2] ... [dirn]
Si el directorio que pretende eliminar contiene ficheros o directorios, el comando devuelve un error. 
Por lo tanto, el directorio no debe contener ni ficheros ni directorios, y ello aunque los propios subdirectorios estén vacíos.
No hay parámetro -r (recursividad) para el comando rmdir.
Para suprimir una estructura tendrá que utilizar el comando rm.
$ rmdir Documentos/Fotos
$ rmdir Documentos/
