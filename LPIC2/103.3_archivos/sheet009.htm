mkdir
El comando mkdir (make directory) permite crear uno o varios directorios, o una estructura completa.
mkdir [-p] dir1 [dir2] ... [dirn]
Por defecto, el comando no crea una estructura. 
Si pasa como argumentos dir1/dir2 y dir1 no existe, el comando devuelve un error. 
En este caso, utilice el parámetro -p.
$ mkdir Documentos
$ mkdir Documentos/Fotos
$ mkdir -p Archivos/antiguallas
