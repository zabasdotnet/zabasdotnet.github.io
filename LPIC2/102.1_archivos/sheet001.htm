102.1 W2 Design hard disk layout
/ (root) El sistema de ficheros raiz / está en la misma partición donde se encuentra el directorio /root.
/var Generalmente actúa como un registro del sistema.
/home Contiene los directorios de trabajo de todos los usuarios, excepto el del superusuario (administrador, root). 
/boot Archivos cargadores de arranque (por ejemplo, los núcleos, el initrd).  A menudo en una partición o disco aparte.
swap space Se usa cuando la cantidad total de memoria RAM está llena y el sistema necesita más recursos.
mount points Directorios accesibles donde se montan dispositivos.
partitions Una partición es una división lógica del disco.
