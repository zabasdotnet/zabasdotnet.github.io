INSERT Volver
Inserte registros en una tabla SQL con el comando INSERT.
INSERT INTO table (campo1, campo2, etc.) VALUES (’valor1’, ’valor2’...);
Por ejemplo, veamos cómo insertar un nuevo cliente en la tabla t_usuarios:
INSERT INTO t_usuarios (id, apellido, nombre, ciudad, id_alias) VALUES (’3’, ’Aguilera’, ’Julio’, ’Olmo’, NULL);
