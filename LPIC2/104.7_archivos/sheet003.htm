El comando locate busca un fichero según el modelo dado en una base de datos de ficheros construida por el comando updatedb. $ locate pepito
/opt/kde3/share/apps/ksgmltools2/docbook/xsl/html/auptotoc.xsl
El comando updatedb recorre una serie de rutas en la cual va a efectuar un find y almacenar todos los resultados en una base indexada. /opt/kde3/share/apps/ksgmltools2/docbook/xsl/params/autotoc.label.separator.xml
/usr/share/gnome/help/gnome-doc-xslt/C/db2html-autotoc.xml
Esto evita, por lo tanto, efectuar de nuevo un find para las búsquedas clásicas. /usr/share/xml/docbook/stylesheet/nwalsh/1.73.1/fo/autotoc.xsl
/usr/share/xml/docbook/stylesheet/nwalsh/1.73.1/html/autotoc.xsl
En la práctica, basta con pasarle a updatedb la lista de las rutas o no incluir los ficheros en la base indexada. /usr/share/xml/docbook/stylesheet/nwalsh/1.73.1/params/autotoc.label.in.hyperlink.xml
/usr/share/xml/docbook/stylesheet/nwalsh/1.73.1/params/autotoc.label.separator.xml
contrab suele lanzar updatedb a diario.
