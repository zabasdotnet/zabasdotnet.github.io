El comando updatedb se encarga de actualizar la base de datos que utiliza locate para buscar ficheros.
Utiliza el fichero con las opciones configuradas en el fichero /etc/updatedb.conf para realizar la busqueda.
El fichero de configuración /etc/updatedb.conf contiene las siguientes opciones:
PRUNE_BIND_MOUNTS : 
Si el valor es 1 o yes los montajes de tipo bind no son incluidos por updatedb.
PRUNEFS : 
Lista de los tipos de sistemas de ficheros que no deben ser actualizados por updatedb.
PRUNENAMES : 
Lista de nombres de directorios (sin rutas absolutas) que no deben ser actualizados por updatedb.
PRUNEPATHS : 
Lista de nombres de directorios (incluidas rutas absolutas) que no deben ser actualizados por updatedb.
Cada ruta debe tener exactamente el mismo nombre que la que aparece cuando se ejecuta el comando locate.
