104.7 W2 Find system files and place files in the correct location
find Busca ficheros según un criterio.
locate Busca un fichero según el modelo dado en una base de datos de ficheros construida por el comando updatedb.
updatedb Recorre una serie de rutas en la cual va a efectuar un find y almacenar todos los resultados en una base indexada.
whereis Encuentra ejecutables.
which Encuentra ejecutables.
type Puede distinguir un comando interno de un comando externo con la ayuda del comando interno type.
/etc/updatedb.conf Fichero de opciones del comando updatedb que se encarga de actualizar la base de datos que utiliza locate.
