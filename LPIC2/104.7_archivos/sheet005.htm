El comando which busca un comando en el PATH (ruta de los ejecutables) y le facilita el primero que encuentra:
$ which date
/bin/date
En Linux hay comandos con el mismo nombre en varias rutas.
Debe precisar el parámetro -a para que which siga su búsqueda.
Sin embargo, si usted ejecuta el comando sin especificar una de esas rutas, se ejecutará el primero por defecto.
$ which -a passwd
/usr/bin/passwd
/usr/bin/X11/passwd
