210.1 W2 DHCP configuration Volver
Candidates should be able to configure a DHCP server.
This objective includes setting default and per client options, adding static hosts and BOOTP hosts.
Also included is configuring a DHCP relay agent and maintaining the DHCP server.
dhcpd.conf Aspectos más básicos de la configuración de un servidor DHCP ISC.
/var/log/daemon.log and /var/log/messages Ficheros de control de errores DHCP.
dhcpd.leases Conserva la información de cada una de las concesiones asignadas por el servidor DHCP ISC.
arp Permite obtener direcciones MAC en caso de querer reservar una IP para la misma tarjeta de red, ejemplo, impresora de red!
dhcpd DEMONIO del servidor DHCP ISC.
