uptime Volver
Este comando permite saber desde hace cuánto tiempo el sistema está funcionando y también obtener información sobre la carga del procesador y el número de usuarios conectados.
*Ejemplo:
root@alpha:~# uptime  
 11:05:51 up  3:59,  3 users,  load average: 0,08, 0,12, 0,13  
root@alpha:~#
Los tres últimos valores indican el promedio de procesos que esperan un procesador, desde 1, 5 y 15 minutos.
Esta información permite constatar una anomalía, si se da el caso de que sean muy elevados o si aumentan progresivamente.
Hay que ponderarlos en función del número de procesadores; un valor de 4 es normal para un procesador de cuatro núcleos, pero no para un procesador de un núcleo.
