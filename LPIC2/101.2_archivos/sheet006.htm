initramfs
Abreviatura de "initial RAM file system", es el sucesor de initrd.
Es un archivo cpio del sistema de ficheros inicial que se ha cargado en memoria durante el proceso de arranque de Linux.
El kernel Linux se monta como un sistema de ficheros raíz y comienza el proceso de inicio desde él.
Esto completará ciertas tareas antes que el sistema de ficheros raíz real se carge.
intramfs necesita incluir todos los drivers de los dispositivos y herramientas necesarios para montar el sistema de ficheros raíz real.
Aparece cuando no se ha compilado bien el kernel y no arranca el sistema
