101.2 W3 Boot the system
dmesg Permite recuperar los mensajes del núcleo emitidos en el momento del arranque de la máquina y los emitidos después.
BIOS Basic Input/Output System (BIOS), también conocido como System BIOS, ROM BIOS o PC BIOS, es un estándar de facto que define a una interfaz de firmware.
bootloader Gestor de arranque
kernel La parte que se ejecuta en modo privilegiado (conocido también como modo núcleo).
initramfs Abreviatura de "initial RAM file system", es el sucesor de initrd.
init El programa init es el primer proceso iniciado y el último que se para dentro del sistema
SysVinit SysVinit está en desuso. Es necesario actualizar a systemd
systemd El reemplazo de init systemd es fruto de una reflexión profunda sobre la ejecución de servicios, combinando todos los sistemas operativos.
