207.1 W3 Basic DNS server configuration Volver
Candidates should be able to configure BIND to function as a caching-only DNS server.
This objective includes the ability to managing a running server and configuring logging.
/etc/named.conf Archivo de configuración que se usará en la carga de named en  modo chroot
/var/named/ Directorio que almacena archivos de declaración de zona.
/usr/sbin/rndc Comando de control BIND que permitetransmitir al servidor una gran variedad de instrucciones.
kill Manda señales especiales a los procesos.
host Herramienta para realizar peticiones DNS en modo no interactivo. 275
