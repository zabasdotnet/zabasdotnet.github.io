ldconfig
El comando ldconfig:
- actualiza la caché para las rutas definidas en /etc/ld.so.conf y asociadas, así como para /usr/lib y /lib;
- actualiza los vínculos simbólicos en las librerías;
- permite también listar las librerías conocidas en la caché.
v --> Modo verboso: indica lo que ldconfig efectúa
N --> No vuelve a construir la caché
N --> No vuelve a construir la caché
Para listar las librerías conocidas del editor de vínculos:
# ldconfig -p
Para saber lo que haría Idconfig pero sin actualizar nada:
# ldconfig -N -X -v
Para actualizar y ver el resultado:
# ldconfig -v
