102.3 W1 Manage shared libraries
ldd El comando ldd permite determinar cuáles son las librerías vinculadas a un programa, y también si están presentes o no.
ldconfig Actualiza el caché de las librerías.
/etc/ld.so.conf Este fichero contiene la lista de los directorios que contienen las librerías compartidas
LD_LIBRARY_PATH Variable que almacena rutas de liberías
