/etc/X11/xorg.conf El fichero de configuración de Xorg está en /etc/X11 y se llama xorg.conf: /etc/X11/xorg.conf.
xhost El comando xhost permite controlar los permisos de acceso al servidor X.
DISPLAY La palabra mágica en un sistema de ventanas X es DISPLAY.
Consiste en lo siguiente:
- un teclado, un ratón y una pantalla
xwininfo Este comando devolverá la información de una ventana especificada.
xdpyinfo xdpyinfo es una utilidad para mostrar información sobre un servidor X.
X X Window System (o X a secas) es una interfaz gráfica desarrollada por el MIT, después por el X Consortium, después por The Open Group y actualmente por X.org.
El nombre en inglés se traduce como “Sistema de ventanas X”

X tiene características bastante interesantes:
- Multiplataforma
- Diseñado para redes
- Flexible
