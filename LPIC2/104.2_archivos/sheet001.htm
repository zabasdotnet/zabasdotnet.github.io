104.2 W2 Maintain the integrity of filesystems
du El comando du (disk usage) facilita la información relativa al espacio ocupado por una estructura (un directorio y todo su contenido).
df El comando df permite obtener estadísticas de ocupación de cada sistema de ficheros montado.
fsck El comando fsck permite comprobar y arreglar un sistema de ficheros.
e2fsck Permite escanear el sistema de archivos de particiones ext2/ext3/ext4 verificando que no existan errores.
mke2fs Se usa para crear sistemas de archivos ext2, ext3, y ext4
debugfs Cambiar opciones del sistema de ficheros.
dumpe2fs Muestra información detallada de sistemas de ficheros.
tune2fs El comando tune2fs permite modificar algunos parámetros de un sistema de ficheros ext2 o ext3
XFS tools Herramientas para administrar filesytems XFS
