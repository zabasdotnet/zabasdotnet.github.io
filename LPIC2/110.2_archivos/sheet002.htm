/etc/nologin Volver
Antes de pasar por un seudoshell de conexión, los módulos PAM permiten aplicar numerosas limitaciones.
Entre ellas, el módulo pam_nologin, que le permite prohibir la conexión de los usuarios excepto root.
Este módulo monitoriza si un usuario intenta conectarse al fichero /etc/nologin.
Es útil para tareas de mantenimiento, ya que sólo root se puede conectar.
$ pwd
/etc/pam.d
$ grep nologin *
login:auth       requisite      pam_nologin.so
ppp:auth         required       pam_nologin.so
sshd:auth        requisite      pam_nologin.so
Piense también que puede prohibir el acceso de una cuenta dada vía el módulo PAM pam_listfile.
