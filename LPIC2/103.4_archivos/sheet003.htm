Tenemos un directorio con varios subdirectorios
Queremos copiar los archivos *.log de esas ubicaciones varias a otra. 
En principio se nos ocurre hacer un find *.log y ir haciendo cp's fichero a fichero. 
Pero si utilizamos:
find /var -name '*.log' | xargs -i cp {} /destino
/var es en este ejemplo el contenedor dónde buscar
-name nombre de fichero
'*.log' la terminación del fichero
-i replace string
{} string
