202.2 W4 System Recovery Volver
Candidates should be able to properly manipulate a Linux system during both the boot process and during recovery mode.
This objective includes using both the init utility and init-related kernel options.
Candidates should be able to determine the cause of errors in loading and usage of bootloaders.
GRUB version 2 and GRUB Legacy are the bootloaders of interest.

Key Knowledge Areas:
- GRUB version 2 and Legacy
- Grub shell
- Boot loader start and hand off to kernel
- Kernel loading
- Hardware initialization and setup
- Daemon/service initialization and setup
- Know the different boot loader install locations on a hard disk or removable device
- Overwriting standard boot loader options and using boot loader shells
- Awareness of UEFI
