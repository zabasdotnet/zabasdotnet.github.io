212.4 W3 Security tasks Volver
Candidates should be able to receive security alerts from various sources, install, configure and run intrusion detection systems and apply security patches and bugfixes.
telnet Acceso en modo terminal
nmap Herramamienta de escaneo de red
fail2ban Herramienta que observa los intentos de login de variados servicios, tales como SSH, FTP, SMTP, HTTP...
Si encuentra intentos de login fallidos una y otra vez desde una misma IP, fail2ban rechazará estos intentos de login bloqueando con reglas de iptables a esas IPs que estaban intentando.
nc Herramienta de red TCP/UDP.
iptables Políticas de firewall.
iptables Filtrado de paquetes.
iptables Gestión de NAT.
iptables Scripts de configuración de reglas de filtrado.
