212.3 W4 Secure shell (SSH) Volver
Candidates should be able to configure and secure an SSH daemon.
This objective includes managing keys and configuring SSH for users.
Candidates should also be able to forward an application protocol over SSH and manage the SSH login.
ssh Apertura de sesion interactiva en servidor SSH.
sshd Demonio del servidor SSH.
/etc/ssh/sshd_config Archivo de configuración del servidor SSH.
/etc/ssh/ Directorio de instalación del servidor SSH.
Private and public key files id_rsa, id_rsa.pub
PermitRootLogin Especifica si root puede acceder al servidor SSH
PubKeyAuthentication Especifica si se podrán usar llaves públicas para demostrar la autenticidad de un usuario.
AllowUsers Permite el acceso o no de usuarios, grupuos, usuarios de otras máquinas, etc...
PasswordAuthentication Permite o no la autenticación de la clave pública
Protocol Versión de protocolo SSH.
Volver Volver Volver
