El comando netstat permite obtener una multitud de información sobre la red y los protocolos.
El parámetro -i permite obtener el estado de las tarjetas de redes con el fin de determinar una posible avería o un problema de cable:

lo   16436   0    1109      0      0      0    1109      0      0  0 LRU

El parámetro -a permite visualizar todas las conexiones, para todos los protocolos, incluido los puertos en escucha de la máquina.
La salida es demasiado larga para reproducirla en estas páginas.
# netstat -a | wc -l
495
