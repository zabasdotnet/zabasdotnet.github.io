cut
El comando cut permite seleccionar columnas y campos en un fichero.
La sintaxis para las columnas es la siguiente:
cut -cColumnas [fic1...]
Una columna es la posición de un carácter en la línea. 
El primer carácter es la columna 1; el segundo, la columna 2, y así sucesivamente. 
Una línea de 80 caracteres dispone de 80 columnas.
La numeración empieza a 1.
El formato de selección de columna es el siguiente:
- una columna sola (p. ej. c2 para la columna 2);
- un intervalo (p. ej. c24 para las columnas 2, 3 y 4);
- una lista de columnas (p. ej. c1,3,6 para las columnas 1, 3 y 6);
- los tres a la vez (p. ej. c13,5,6,12).
El comando cut también permite seleccionar campos.
Se deben delimitar estos campos por defecto por una tabulación, pero el parámetro -d permite seleccionar otro carácter (espacio, ;). 
El carácter separador debe ser único.
No es posible poner ni dos, ni tres, ni una cadena de separadores.
Para eliminar los caracteres múltiples, utilice tr.
El separador por defecto es la tabulación.
Por defecto se sustituyen las tabulaciones con espacios dentro de los editores...
cut -dc -fCampos [fic1...]
Si no hay delimitador (tabulación u otro) en una línea, cut muestra toda la línea.
