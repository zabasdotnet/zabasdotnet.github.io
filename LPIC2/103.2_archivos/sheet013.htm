tail
Comando para ver las últimas líneas de un fichero.
tail [+/-valor[b/c]] [-f] [fic1...]
Al igual que para head, por defecto se visualizan las 10 últimas líneas.
El valor -numlíneas permite modificar este estado.
Use c para indicar un número de caracteres. 
Una b indica un número de bloques (512 bytes por bloque).
Un + invierte el orden del comando y se convierte en un head (tail +10 <=> head -n 10).
Finalmente, la opción -f deja el fichero abierto. 
Si se inserta más información en él (por ejemplo, un fichero de registros), se visualizará su contenido en tiempo real en la pantalla hasta que el usuario lo interrumpa de manera voluntaria ([Ctrl] C).
$ tail -5 lista
$ tail -10c lista"
