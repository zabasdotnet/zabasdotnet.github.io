103.2 W3 Process text streams using filters
cat Visualiza texto en pantalla completa en bloque
cut El comando cut permite seleccionar columnas y campos en un fichero.
expand El comando expand convierte las tabulaciones en espacios.
fmt Formatea cada párrafo, uniendo o separando líneas para que todas tengan el mismo tamaño
head Para ver el principio del contenido de un fichero.
join El comando join permite efectuar la unión de dos ficheros en función de un campo común.
less Muestra la salida estándar o el contenido de un fichero de forma paginada. $ less fichero
nl Numerar las líneas, lo mismo que cat -n
od El comando od (Octal Dump) nos permite ver byte a byte el contenido de un archivo.
paste El comando paste agrupa n ficheros en uno.
pr pr convierte ficheros de texto para imprimirlos.
sed sed es un editor de flujo (Stream Editor) que permite filtrar y transformar texto. 
sort El comando sort permite ordenar las líneas.
split split permite recortar un gran fichero en varios trozos con un tamaño determinado.
tail Comando para ver las últimas líneas de un fichero.
tr El comando tr permite sustituir unos caracteres con otros y sólo acepta datos que provengan del canal de entrada estándar, no de los ficheros.
unexpand El comando unexpand convierte los espacios en tabulaciones. Con el parámetro -a sustituye todas las secuencias de al menos dos espacios por el número necesario de tabulaciones.
uniq El comando uniq permite suprimir las líneas repetidas en flujos en entrada o ficheros ordenados.
wc El comando wc (word count) permite contar las líneas, las palabras y los caracteres.
