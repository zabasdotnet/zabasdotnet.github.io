102.5 W3 Use RPM and YUM package management
rpm Gestor de paquetes Red Hat
rpm2cpio Convierte  los contenidos en un archivo cpio.
/etc/yum/conf Fichero de configuración principal de YUM donde se colocan los repostiorios.
/etc/yum.repos.d/ Directorio que almacena ficheros de configuración YUM
yum Un programa de gestión de paquetes tipo RPM.
yumdownloader Mediante la utilidad yumdownloader podemos descargar un paquete de los repositorios instalados en el sistema.
Para poder disponer de la utilidad yumdownloader deberemos tener instalado el paquete yum-utils.		 # yum install yum-utils -y

# yumdownloader ntp
