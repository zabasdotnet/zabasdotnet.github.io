209.2 W3 NFS Server Configuration Volver
Candidates should be able to export filesystems using NFS.
This objective includes access restrictions, mounting an NFS filesystem on a client and securing NFS.
/etc/exports Para declarar una compartición permanente activa del servicio NFS.
exportfs Especifica las opciones de comparticion dinámicamente
showmount Muestra la información de un servidor NFS remoto.
nfsstat Sirve para verificar una actividad o una ausencia de actividad en un servidor NFS.
/proc/mounts Muestra la información de una compartición NFS.
/etc/fstab Se define la información de una compartición NFS.
rpcinfo Efectua una petición RPC a un servidor y muestra los daemons gestionados.
mountd DEMONIO que gestiona las peticiones de montaje de los clientes.
portmapper Peticiones RPC (Remote Procedure Call).
