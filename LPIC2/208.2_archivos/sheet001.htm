208.2 W3 Apache configuration for HTTPS Volver
Candidates should be able to configure a web server to provide HTTPS.
Apache2 configuration files
/etc/ssl/, /etc/pki/
openssl, CA.pl Librería para la generación de certificados.
SSLEngine Activación del motor SSL
SSLCertificateKeyFile Define el archivo que contiene la clave privada del servidor.
SSLCertificateFile Define el archivo que contiene el certificado del servidor
SSLCertificateChainFile Define el archivo que contiene el certificado desde root hasta el del usuario final.
SSLCACertificateFile Define el archivo de la autentificación del certificado.
SSLCACertificatePath Define la ruta del archivo de la autentificación del certificado.
SSLProtocol Define el protocolo SSL (SSLv3, TLSv1, TLSv1.1, TLSv1.2)
SSLCipherSuite Define los métodos de cifrado (SSLCipherSuite HIGH:MEDIUM:!ADH)
ServerTokens Determina que información mostrará el servidor en la cabezera de respuesta HTTP. (PROD)
ServerSignature Aparece en el pie de página como 404, listado de directorios, etc.
TraceEnable Peticiones de traza HTTP (ACTIVO por defecto).
