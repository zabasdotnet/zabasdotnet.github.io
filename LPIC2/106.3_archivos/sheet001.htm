Sticky/Repeat Keys XFree86 soporta una opción de accesibilidad que permite a los usuarios discapacitados teclear con una sola mano.
Con las "sticky keys" acticvadas, el usuario puede pulsar una tecla de modificación (ctrl, alt, shift) seguida de otra tecla, mejor que tener que dejar pulsada la tecla de modificación mientras pulsa la letra.

Para activar las "sticky keys", primero hay que asegurar que la extensión xkb está activada (esto se hace durante la configuración inicial del servidor X y suele estar activado por defecto).
Después, ejectuar el servidor X con la opción +accessx.
Si usamos startx, ya sea ejecutando startx -- +accessx o añadiendo +access a la linea de argumentos de servidor en el script de startx.
Si usamos xdm, hay que añadir +accessx a la línea apropiada de servidor en /etc/X11/xdm/Xservers.
También es posible activar la accesibilidad X con alguna ultilidad de usuario final mientras se ejecutar el servidor X.
Una vez que la accesibilidad X es activada, pulsando la tecla shift 5 veces seguidas se activan las sticky keys.
Para desactivar las sticky keys, o bien pulsamos la tecla shift 5 veces seguidas de nuevo, o pulsamos una tecla mientras mantenemos pulsada una tecla de modificación.

Mouse keys Podemos utilizar las Mouse keys (teclas de ratón) para controlar el ratón con el teclado.

Para activar/desactivar las Mouse keys tenemos que pulsar la siguiente combinación de teclas:
Alt + Shift + Bloq.Num.

Una vez activadas podemos controlar el ratón desde el teclado numérico de nuestro equipo de la siguiente forma:
Todos los números menos el 5 nos permiten mover el ratón.
Con el 5 hacemos clic, según el botón del ratón que tengamos seleccionado con / (izquierdo), * (central) y – (derecho), y con el + simulamos el doble clic.
Por último, con 0 presionamos el botón y con . lo liberamos.

emacspeak EmacSpeak es un completo escritorio de audio y voz para Emacs.
Emacspeak implementa muchas técnicas modernas para leer el contenido de los búferes de Emacs por medio de sintetizadores de voces para usuarios ciegos o discapacitados visuales o de otro tipo.

Emacspeak puede (y lo hace) implementar la habilitación de la voz para casi todos los modos mayores y menores de uso extendido.

Esto no se limita a los ciegos.
Cualquier usuario que sepa aprovecharlo, dispondrá de un amplio espectro de posibilidades.

Existen varios sintetizadores de voz basados en software que se pueden usar con Emacspeak.
También hay hardware especial.
