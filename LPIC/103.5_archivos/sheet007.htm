Si cierras, apagas o reinicias tu maquina o terminal, la terminal virtual sigue ejecutándose junto con cualquier proceso que hayas puesto a ejecutar dentro de esta, por ejemplo un respaldo, descarga, actualización etc.
screen -ls --> Para ver las terminales que se están ejecutado.
screen -r --> acompañado del numero id que muestra para entrar a la terminal virtual que deseas.
screen -r --> Recupera la única terminal si no hay otras.
Para cerrar correctamente la terminal virtual ejecuta dentro de esta el comando exit
