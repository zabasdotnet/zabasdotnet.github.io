103.1 W4 Work on the command line
bash Es una interfaz que funciona en modo texto entre el núcleo de Linux y los usuarios (avanzados) o las aplicaciones.
echo El comando echo está hecho para visualizar texto.
env Muestra las variables de usuario y de sistema, nombre y contenido.
export El comando export permite exportar una variable de manera que su contenido sea visible por los scripts y otros subshells.
pwd Devuelve el directorio actual.  Con la opción -P, el directorio físico sin enlaces simbólicos. Con la opción -L para los enclaces simbólicos.
set Muestra las variables de entorno
unset Este comando borra una o más variables y si es un array no afecta al resto de elementos.
man El programa man es una herramienta que se utiliza para documentar y aprender sobre comandos, archivos, llamadas de sistema, etc. 
uname El comando uname (unix name) permite obtener toda la información correspondiente a la versión de Unix de manera precisa y completa.
history Puede ver el contenido del historial con el comando history. 
.bash_history El shell conserva así un gran número de entradas en el historial (se puede modificar el número de líneas guardadas).  Se conserva este historial en un fichero caché de su directorio personal llamado .bash_history.
