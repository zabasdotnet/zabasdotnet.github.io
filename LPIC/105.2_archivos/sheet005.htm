if ... then ... else
La estructura if then else fi es una estructura de control condicional.
if <comandos_condición>
   then
   <comandos ejecutados si condición cumplida>
   else
   <comandos ejecutados si última condición no realizada>
fi
También puede especificar elif.
En realidad es un else if.
Si no se cumple la última condición, se prueba una nueva.
El comando case ... esac permite comprobar el contenido de una variable o de un resultado de manera múltiple.
case Valor in
   Modelo1)
      Comandos ;;
   Modelo2)
      Comandos ;;
   *) accion_defecto ;;
esac
El modelo es o bien un texto simple o bien uno compuesto de caracteres especiales.
Cada bloque de comandos relacionados con el modelo se debe terminar con dos puntos y coma.
En cuanto se haya comprobado el modelo, se ejecuta el bloque de comandos correspondiente.
El asterisco en la última posición (cadena variable) es la acción por defecto si no se establece ningún criterio; es opcional.
* Cadena variable (incluso vacía)
? Un solo carácter
[...] Un intervalo de caracteres
[!...] Negación del intervalo de caracteres | O lógico
