104.1 W2 Create Partitions and filesystems
fdisk Herramienta de gestión de particiones.
gdisk Herramienta de gestión de particiones GPT
parted parted permite operaciones muy avanzadas en las particiones, como, por ejemplo, su redimensionamiento. Presenta una interfaz interactiva (intérprete de comandos) que atiende a scripts.
hay interfaces gráficas de parted, como qtparted o gparted.
Se pueden usar los comandos de parted como mkpart, rm o resize para trabajar con las particiones.
El comando partx también permite notificar al núcleo las modificaciones realizadas a las tablas de partición.
mkfs Crear un sistema de ficheros. mkfs llama a otros programas en función del tipo de sistema de ficheros seleccionado.
