El comando mke2fs es usado para crear el sistema de archivos de Linux ext2 en un dispositivo tal como una partición de disco duro o un disquete.
mke2fs esencialmente formatea el dispositivo y crea un dispositivo vacío compatible con Linux que puede luego ser usado para almacenar archivos y datos.
/sbin/mke2fs /dev/fd0
La utilidad mke2fs tiene varias opciones.
La opción -c hace que el comando mke2fs verifique el dispositivo por sectores dañados antes de crear el sistema de archivos.
