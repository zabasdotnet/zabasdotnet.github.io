apt-get
apt-get es la herramienta de línea de comandos que se suministra con el paquete Debian apt.
APT busca los paquetes en su lista cacheada y muestra las depencendias que deben ser instaladas o actualizadas.
APT recupera, configura e instala las dependencias automáticamente.
-> update --> se usa para resincronizar los ficheros indices de los orígenes
-> upgrade --> se usa para instalar las veriones nuevas de todos los paquetes actualmente instalados en el sistema.
-> dist-upgrade --> se usa para actualizar todo el sistema (lo mismo que aptitude full-upgrade), ya que la opción upgrade únicamente intenta hacerlo con los más importantes.
El comando apt-get install xxx instala el paquete xxx:
Dos opciones merecen recordarse:
la -s para la simulación: APT indica lo que debería hacer, pero no lo hace.
la -f para «fix-broken»: APT intenta arreglar los problemas de dependencias como puede (añadido de paquetes).
La herramienta synaptic es un front-end: una interfaz gráfica que llama a las funciones de APT.
Soporta todas las operaciones propuestas por APT y, además, disfruta de una interfaz muy atractiva.
Synaptic es un front-end para APT.
