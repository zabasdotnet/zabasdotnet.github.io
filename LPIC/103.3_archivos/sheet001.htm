103.3 W4 Perform basic file management
cp El comando cp (copy) copia uno o varios ficheros en otro fichero o en un directorio
find Búsqueda de ficheros por criterios
mkdir El comando mkdir (make directory) permite crear uno o varios directorios, o una estructura completa.
mv El comando mv (move) permite mover, volver a nombrar un fichero o las dos cosas a la vez. 
ls El comando ls permite listar el contenido de un directorio (catálogo) en líneas o columnas.
rm El comando rm (remove) suprime uno o varios ficheros y, si es preciso, una estructura completa.
rmdir El comando rmdir (remove directory) suprime uno o varios directorios. 
touch Puede necesitar crear ficheros vacíos para hacer pruebas
tar Crea archivos de los ficheros, incluida la estructura de ficheros, en cualquier tipo de soporte, incluido en otro fichero (archivo con la extensión .tar).
cpio El comando cpio hace una copia de seguridad sobre la salida estándar de los ficheros.
dd Para la copia de seguridad física de discos y sistemas de ficheros (los dumps), utilice el comando dd.
file El comando file permite diferenciar los tipos de ficheros.
gzip Compresión en formato GnuZip.
gunzip Descompresión en formato GnuZip.
bzip2 Compresión al formato bzip2.
xc
file globbing Búsqueda, en la línea de comandos, de algunos caracteres especiales (metacaracteres), con la finalidad de “interpretarlos”.
