210.4 W4 Configuring an OpenLDAP server Volver
Candidates should be able to configure a basic OpenLDAP server including knowledge of LDIF format and essential access controls.
An understanding of the role of SSSD in authentication and identity management is included.
slapd Nombre del ejecutable del servicio OpenLDAP.
slapd.conf Archivo de configuración (generalmente ubicado en el directorio /etc/ldap o /etc/openldap)
LDIF LDIF (LDAP Data Interchange Format - Formato de intercambio de datos LDAP) tiene como objetivo permitir la exportación o la importación de datos desde o hacia un directorio LDAP.
slapadd Se usa para añadir ojetos desde un fichero LDIF a una base de datos splapd (Stand-alone LDAP Daemon).
slapcat Se usa para generar un fichero LDIF basado en los contenidos de la base de datos slapd (Stand-alone LDAP Daemon).
slapindex Se usa para regenerar los indices de la base de datos slapd (Stand-alone LDAP Daemon).
/var/lib/ldap/ Directorio por defecto de OpenLDAP.
loglevel OpenLDAP soporta un registro de errores muy detallado y configurable con su directiva loglevel (olcLogLevel for the cn=config configuration method).
