Cuando se crea, un fichero o un directorio dispone de permisos por defecto.
Utilice el comando chmod (change mode) para modificar los permisos en un fichero o un directorio.
Existen dos métodos para modificar estos derechos:
- mediante símbolos
- mediante un sistema octal de representación de permisos.
Sólo el propietario de un fichero puede modificar sus permisos (más el administrador del sistema).
El parámetro -R cambia los permisos de manera recursiva.
