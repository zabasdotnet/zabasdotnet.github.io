UPDATE Volver
La instrucción UPDATE actualiza uno o varios registros de una tabla.
 
La cláusula WHERE es opcional. 
En el ejemplo, se actualizan todos los registros.
UPDATE tabla SET campo1=’valor’, campo2=’valor’ [WHERE campon=’valor’]
Veamos cómo aumentar todos los precios un 5%:
UPDATE t_productos SET precio=precio*1.05;
